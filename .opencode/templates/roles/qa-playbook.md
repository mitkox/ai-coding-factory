# QA/Test Playbook

## Purpose

Ensure every story has automated tests, coverage thresholds are met, and traceability is complete.

## Responsibilities

- Validate test linkage to story IDs
- Enforce coverage thresholds (domain + application)
- Run integration and smoke tests
- Review release readiness checklist

## Required Artifacts

- Test plan or notes
- Traceability report
- Coverage summary
